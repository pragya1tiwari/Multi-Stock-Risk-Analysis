# Multi-Stock Risk and Correlation Analysis

## Overview

This project analyzes historical stock market data from multiple technology companies to study price trends, returns, volatility, and correlations using Python.

## Objective

The objective of this project is to apply quantitative finance concepts to understand relationships between assets and evaluate market behavior using historical data.

## Stocks Analyzed

- Apple (AAPL)
- Microsoft (MSFT)
- Alphabet (GOOG)
- Amazon (AMZN)

## Tools and Libraries

- Python
- Pandas
- Matplotlib
- Seaborn
- yfinance
- Jupyter Notebook

## Methodology

### 1. Data Collection
Historical stock prices were downloaded using the Yahoo Finance API.

### 2. Price Trend Analysis
Stock price movements were visualized and compared over time.

### 3. Return Analysis
Daily percentage returns were calculated to measure market performance.

### 4. Volatility Estimation
Risk was estimated using the standard deviation of daily returns.

### 5. Correlation Analysis
Correlation matrices and heatmaps were used to examine relationships among stocks.

## Key Findings

- Technology stocks exhibited strong positive correlations.
- Volatility differed across companies despite operating in similar sectors.
- Correlation analysis provided insights into diversification and portfolio construction.
- Daily returns revealed short-term market fluctuations that are not visible in price charts alone.

## Skills Demonstrated

- Financial Time-Series Analysis
- Data Visualization
- Statistical Analysis
- Risk Modeling
- Correlation Analysis

## Conclusion

This project demonstrates foundational quantitative finance techniques including return analysis, volatility estimation, and correlation modeling using Python.
